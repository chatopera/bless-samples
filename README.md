# bless-samples

Node.js packaging utility to hide your source code.

## Install

```
npm install -g @chatopera/bless
```

## Compile

- basic sample

```
cd projects
bless dist -p purejs
cd purejs/bless-gen/dist
npm install
node app.js
```

- advanced sample

```
cd projects
bless dist -p complex
cd complex/bless-gen/dist
npm install
node app.js
```
